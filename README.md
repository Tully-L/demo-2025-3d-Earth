### ✨ Features

- Realistic Earth rendering with day/night textures
- Dynamic cloud layer with adjustable opacity
- Atmospheric scatter effect
- Smooth rotation and interaction
- Responsive design for all devices

### 🛠️ Tech Stack

- React 18
- Three.js
- TypeScript
- Tailwind CSS
- Vite

### 🚀 Getting Started

#### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

#### Installation

1. Clone the repository
```bash
git clone https://github.com/Tully-L/demo-2025-3d-Earth.git
cd demo-2025-3d-Earth
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start development server
```bash
npm run dev
# or
yarn dev
```

4. Build for production
```bash
npm run build
# or
yarn build
```

### 📁 Project Structure

```
src/

  │   └── earth-globe.tsx    # Main Earth component
  ├── pages/
  │   └── home.tsx          # Home page
  └── App.tsx               # Root component
```
