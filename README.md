## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher) 📦
- npm or yarn 📥

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Tully-L/0703-demo-campus.git
   cd 0703-demo-campus
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
   Access at `http://localhost:5173`

4. **Build for production**
   ```bash
   npm run build
   # or
   yarn build
   ```
